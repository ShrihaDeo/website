✨Occupancy Detection – Internship✨

During my internship, I contributed to the development of a smart occupancy detection system using a combination of Passive Infrared (PIR) and Time-of-Flight (ToF) sensors. The objective of the project was to accurately detect human presence and occupancy within a defined area to optimize energy usage and environmental control systems.

My role in the project involved hands-on circuit design, where I wired and integrated PIR and ToF sensors to create a reliable detection setup. I also used embedded C to program the microcontroller logic, ensuring the system could process input from both types of sensors to make smart occupancy decisions in real-time.

In addition to the technical implementation, I worked with CAD software to design and 3D-print a protective housing for the system. This casing ensured the sensors were mounted securely and operated efficiently in varied environments. The design process emphasized both functionality and aesthetic integration into modern indoor spaces.

The system featured a Python-based server hosted on a Raspberry Pi, which collected and transmitted sensor data remotely. This data was then pushed to a dynamic Node.js frontend, providing a live visualization of occupancy status. This full-stack pipeline facilitated remote monitoring and data logging, offering real-world insights into space utilization.

Overall, this internship experience deepened my understanding of IoT system development , embedded systems, hardware-software integration, and real-time web communication. It provided a strong foundation in designing intelligent environments using practical sensor technologies and cloud-connected architectures.
