✨Minesweeper✨

A Minesweeper game was developed in Python as a personal project, driven by a strong passion for game coding. The game aims to replicate the classic Minesweeper experience with added features and improvements. The project includes the following key components:

  • Gameplay Replication: Replicates the classic Minesweeper experience with added features and improvements.
  • User Interface Enhancements: Focused on improving the user interface for a better gaming experience.
  • Ongoing Development: The project is a work in progress, with continuous updates to enhance gameplay and overall functionality.

This project serves as an opportunity to refine programming skills and explore game development techniques in Python.
