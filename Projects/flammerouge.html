✨Flamme Rouge✨

This interactive card game is inspired by the Flamme Rouge board game. It was developed in C# as part of a first-year project, focusing on strategic gameplay and player engagement. The project included the following key components:

  • Game Mechanics Design: Designed engaging game mechanics to ensure strategic gameplay and player involvement.
  • Microsoft Visual Studio: Utilized Visual Studio for coding, testing, and debugging, which helped streamline the development process.
  • Player Interaction: Focused on improving user interactions to enhance the overall gameplay experience.

Key programming concepts applied in this project included:

  • Object-Oriented Programming: Used OOP principles to structure the game and its components efficiently.
  • Event Handling: Implemented event-driven programming to handle user inputs and game interactions.
  • User Interface Design: Developed a user-friendly interface to improve gameplay experience.

Through this project, I improved my problem-solving skills, logical thinking, and coding abilities.

In addition to technical skills, this experience gave me a better understanding of game development, software design, and interactive application creation. It was a valuable learning opportunity that strengthened my passion for building engaging and well-structured digital experiences.
