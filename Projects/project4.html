✨Circuit Application✨

An application was developed in C# as a Year 1 project to build and simulate circuits using logical principles. This tool allows users to design custom circuits and test their behavior interactively, reinforcing key computing and engineering concepts.

The project focused on the following core features:

  • Circuit Design: Enabled users to visually create circuits using logical components, supporting better understanding through hands-on exploration.
  • Logic Gates Implementation: Implemented essential logic gates and operations to emulate real-world digital circuits.
  • Simulation: Allowed for real-time simulation of designed circuits, helping users experiment and test logic-based behavior.

This project offered experience in multiple areas of development and computing:

  • C# Programming: Applied core C# development skills to implement UI, logic processing, and simulations.
  • Problem-Solving: Tackled challenges in simulating accurate logic behavior through carefully structured algorithms.
  • Software Design: Developed a functional and educational tool that reinforces digital logic principles for beginners.

This project helped establish a strong foundation in logic, programming, and simulation-based learning tools, setting the stage for more advanced development work in the future.
